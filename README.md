#  React E-commerce App

A modern e-commerce web application built with Next.js and React. Users can browse products, add items to cart, manage a wishlist, and interact with a clean, responsive UI.

##  Features

* Product listing with search, filtering, and sorting
* Shopping cart with quantity management
* Wishlist system ❤️
* Persistent cart using localStorage
* Responsive design with Tailwind CSS
* Modern UI with glassmorphism effects

##  Tech Stack

* Next.js (App Router)
* React
* TypeScript
* Tailwind CSS
* Context API (State Management)

##  Getting Started

```bash
npm install
npm run dev
```
